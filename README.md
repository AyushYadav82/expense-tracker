# 💰 Expense Tracker

A simple web-based expense tracker built using **Python (Flask)**.

## 🚀 Features
- Add and view expenses  
- Set starting balance  
- Track total spending & remaining balance  
- Add date, time, and notes  

## 🛠️ Tech Stack
- Python  
- Flask  
- HTML, CSS
